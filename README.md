### Dependencies
* Windows 10
* Express
* Body-parser
* Cors
* Nodemon (optionally)
* Node.js
* Vite

### Installing
* Open a new terminal
* Type in:
```
npm create vite@latest *folderName*
npm install

npm int -y
npm i express mongoose body-parser cors --save
```

### Executing Program
* Open terminal
* Type in: 
```
node app
```
or, if using nodemon:
```
npx nodemon app
```
